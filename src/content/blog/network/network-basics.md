## 计算机的发展史

### 什么是计算机

计算机俗称**电脑**，是一种用于高速计算的电子计算机器，可以进行数值计算，又可以进行逻辑计算，具备存储记忆的功能，是可以按照程序进行，自动，高速进行海量数据的现代化智能电子设备。

- **超级计算机**：气象预测、核能模拟、基因分析、人工智能训练等。
- **大型计算机**：银行交易处理、政府数据中心、航空订票系统等。
- **服务器**：为其他设备（客户端）提供网络服务（如数据存储、网站托管）。
- **小型机**：企业资源管理（ERP）、科学计算等。
- **工作站**：高性能个人计算机，专为专业领域设计（如图形渲染、工程设计）。
- **个人计算机**：面向个人用户，包括台式机、笔记本电脑、平板电脑等。
- **嵌入式计算机**：集成在设备内部，执行特定功能（如智能家电、汽车控制系统、医疗设备）。

### 计算机的发展历史

1. **早期计算设备（19 世纪 - 20 世纪初）**

- **机械计算机**：“在 19 世纪，像巴贝奇的差分机这样的机械计算机，是那个时代对复杂计算需求的回应，虽然它们很笨重，但奠定了自动计算的基础。”
- **个人观点**：机械计算机虽然受限于物理磨损和制造工艺，但其“程序控制”的思想是现代计算机的灵魂。

2. **电子计算机的诞生（20 世纪中期）**

- **电子管计算机（如 ENIAC）**：“二战期间和战后，电子管计算机的出现，如 ENIAC，标志着计算速度的巨大飞跃，从机械时代的缓慢计算转变为电子速度。”
- **晶体管计算机**：“晶体管的发明极大地提高了计算机的可靠性和效率，缩小了体积，为商业化铺平了道路。”

3. **集成电路与个人计算机革命（20 世纪后期）**

- **集成电路的发展**：摩尔定律驱动了芯片性能的指数级增长。
- **个人计算机的兴起**：Apple II 和 IBM PC 让计算力走进家庭。

4. **互联网与现代计算（21 世纪初至今）**

- **互联网的普及**：全球化数字经济的诞生。
- **移动计算和人工智能**：智能手机与深度学习改变了各行各业。

## 计算机的大脑：CPU

### 什么是 CPU

CPU（Central Processing Unit），即**中央处理器**，是计算机的大脑，负责执行指令和处理数据。

### 指令获取与执行

- **工作过程**：读取指令 -> 解码指令 -> 执行操作 -> 结果输出。
- **个人理解**：CPU 的流程就像解决问题：先理解问题（解码），再思考方案（执行），最后得出答案（输出）。

### 主要功能单元

- **算术逻辑单元（ALU）**：CPU 的心脏，负责所有的算术（加减乘除）和逻辑（判断）运算。
- **控制单元（CU）**：CPU 的指挥部，协调从内存取指令、解码及控制其他部件。
- **寄存器**：CPU 内部的高速存储单元，相当于“临时记忆”，速度远超内存。

### 性能指标

- **主频**：工作频率（GHz），频率越高，单位时间执行指令越多。
- **核心数量**：多核设计就像多条生产线，提高多任务处理能力。
- **缓存 (Cache)**：位于 CPU 与内存之间，减少 CPU 访问慢速内存的次数。
- **架构和指令集**：决定了 CPU 的“语言”和执行效率（如 x86, ARM）。

### CPU 的分类

- **Intel**：以稳定著称，Core（酷睿）系列面向大众，Xeon（至强）系列面向服务器。
- **AMD**：以高性价比和多核性能著称，Ryzen（锐龙）系列深受玩家喜爱。
- **ARM**：低功耗架构，广泛用于手机（苹果 A 系列、骁龙）和嵌入式设备。

> **小贴士**：一般办公选 Intel，游戏/生产力追求极致可选 AMD，但这没有绝对界限，看个人需求。

### 选购与故障排除

- **指标**：看主频、缓存、核数。
- **常见故障**：

1. **散热不良**：死机、蓝屏。需清洁风扇或重涂硅脂。
2. **超频问题**：自检不通过。需恢复 BIOS 默认频率。
3. **针脚接触不良**：无法启动。检查插槽是否有异物或针脚弯曲。

## 计算机的工作台：内存 (Memory)

### 内存简介与原理

内存是用于暂时存储 CPU 正在使用的数据和程序，以供快速访问。

- **原理**：像是有许多“小抽屉”的柜子，每个抽屉有唯一编号（内存地址）。
- **易失性**：断电后数据会消失，所以需要硬盘进行长期存储。

### 内存的分类

- **按功能**：
- **RAM（随机存取存储器）**：可读写，如 DDR4、DDR5。
- **ROM（只读存储器）**：固件存储，通常不可随意修改。

- **按技术**：目前主流是 **DDR5**，传输速率远超前代。

### 选购与维护

- **指南**：关注主板兼容性（插槽类型）、容量（8G 办公，16G+ 游戏）、频率和时序（CL 值）。
- **故障解决**：
- **无法开机**：尝试重新插拔，用橡皮擦擦拭“金手指”。
- **蓝屏**：使用系统自带的内存诊断工具检查。

## 计算机的仓库：硬盘 (Disk)

### 硬盘种类对比

- **机械硬盘 (HDD)**：盘片旋转读写。容量大、便宜，适合存电影/文档。
- **固态硬盘 (SSD)**：闪存芯片存储。速度极快（秒开机）、抗震，适合装系统和软件。

### 常见故障原因

1. **系统不认硬盘**：数据线松动或 BIOS 跳线设置错误。
2. **系统无法启动**：引导文件丢失（可用 PE 系统修复）。
3. **硬盘坏道**：物理坏道建议送修，逻辑坏道可用软件修复。

## 计算机的骨架：主板 (Motherboard)

### 主板的作用

主板连接并支撑所有硬件，负责数据流动的协调。

- **核心组件**：芯片组、BIOS、CPU 插槽、内存插槽、PCIe 插槽。

### 选购要点

- **板型**：ATX（大板）、Micro-ATX（中板）、Mini-ITX（小板）。
- **品牌**：华硕（ASUS）、技嘉（Gigabyte）、微星（MSI）是御三家。
- **扩展性**：USB 接口数量、是否自带 Wi-Fi/蓝牙。

## 计算机的视觉神经：显卡 (GPU)

### 显卡简介

显卡负责将数据转换为显示器能识别的图像信号。在现代前端开发中，GPU 也常用于网页的硬件加速渲染。

- **核心任务**：图像生成、AI 计算、视频解码。

### 显卡接口演进

1. ISA -> 2. PCI -> 3. AGP -> **4. PCI-E** (当前主流)
